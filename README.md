# Registro Básico de Ventas Diarias

## Descripción
Este proyecto es un programa en Python que permite registrar una venta y mostrar un resumen claro en consola.

## Funcionalidades
El sistema solicita al usuario:

- Nombre del cliente
- Precio unitario del producto
- Cantidad de productos comprados
- Confirmación de si el cliente tiene membresía VIP

## Requisitos técnicos

- Uso de variables
- Tipos de datos básicos (int, float, str, bool)
- Entrada de datos por consola
- Salida de información por consola
- Uso de estructuras condicionales (if)

## Resultado esperado
Al finalizar la ejecución, el programa muestra:

- Nombre del cliente
- Subtotal
- Valor del descuento
- Total final a pagar
